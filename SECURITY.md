# Security Policy

## Supported Versions

The below versions are supported.

| Version | Supported          |
| ------- | ------------------ |
| ≥ 0.10.0   | :white_check_mark: |
| 0.9.0   | :white_check_mark:*                |
| < 0.8.x   | :x:                |

* Supported but not recommended for use.

## Reporting a Vulnerability

Please report any security vunerabilities to the upstream packages and their respective repositories. If, in a rare case, the vunerability is directly exploitable through only a method in this code base, please submit a PR or contact me on Discord, based on severity.

## Standards for package security

Do not update packages until at least a day as passed since its publishing. Only allow pre-install and post-install scripts to run from wxt.
